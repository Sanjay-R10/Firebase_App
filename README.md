# Firebase Auth App

A React application with user authentication powered by Firebase.

## Features
- User Login & Registration
- Saves user email and details to Firebase Firestore
- Firebase console shows all registered users and their details in real-time

## Tech Stack
- React (Vite)
- Firebase Authentication
- Firebase Firestore (Database)
- JavaScript

## How It Works
1. User signs up / logs in via the login page
2. Their email and details are saved to Firebase Firestore
3. All user data is visible in the Firebase console backend
